# Proyectos de MPLABX

Este repositorio contiene una colección de proyectos y prácticas desarrolladas con MPLAB X IDE, enfocadas en la programación de microcontroladores PIC.

## Descripción de las Prácticas

A continuación, se detalla el propósito y la funcionalidad de cada proyecto (carpeta `p_n`).

### Práctica 2: p_2

* **Descripción:** Implementa un "espejo" de puertos. El programa lee el valor de entrada del **PORTA** y lo refleja (muestra) en la salida del **PORTB**.

### Práctica 20: p_20

* **Descripción:** Implementa un **contador de década** (0-9) que utiliza el **Timer0** para generar la base de tiempos y actualizar el conteo automáticamente.

### Práctica 21: p_21

* **Descripción:** Implementa un **contador decimal** (0-9) controlado manualmente. El incremento del contador se realiza mediante un **botón** (probablemente usando interrupciones o sondeo).

### Práctica 22: p_22

* **Descripción:** Implementa un **contador de década** (0-9) controlado por un **botón**, similar a la práctica anterior.
